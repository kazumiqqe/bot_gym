#Gym Bot
Telegram бот для помощи в тренировках.
## Возможности

Программы тренировок: несколько готовых программ на выбор
Плейлисты для зала: подборки музыки для тренировок
Учёт прогрессии: запись весов и повторений по каждому упражнению
История упражнений: отслеживай свой прогресс
Совет ИИ: персональные рекомендации по прогрессии нагрузок

## Стек

- Python
- aiogram 3.x
- SQLite
- Groq API (llama-3.3-70b-versatile)

## Запуск

1. Клонируй репозиторий
2. Установи зависимости: `pip install -r requirements.txt`
3. Создай `.env` файл:
```
BOT_TOKEN=твой_токен
CHANNEL_ID=@твой_канал
GROQ_API_KEY=твой_ключ
```
4. Запусти: `python main.py`
